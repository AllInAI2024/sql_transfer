欢迎使用 SQL 脚本转换工具

将基于达梦数据库中间表的可视化脚本，转换为直接查询 ODPS (MaxCompute) 原始表的新指标脚本。

功能模块

📊

可视化脚本

管理和查看可视化脚本，包括脚本的导入、编辑和删除。

🔒

匿名化

对脚本中的敏感信息进行匿名化处理，保护数据安全。

🔄

脚本转换

核心转换功能，将中间表脚本转换为直接查询 ODPS 的脚本。

✅

验证

验证转换后的脚本正确性，确保转换结果符合预期。

⚙️

配置

系统配置管理，包括数据库连接、API 密钥等设置。

转换本质

  • 可视化脚本 查询达梦数据库的中间表（通过中间表脚本从 ODPS 原始表构建）
  • 中间表脚本 定义了中间表如何从 ODPS 原始表获取数据
  • 转换时需要分析中间表脚本，理解 ODPS 原始表的结构和查询逻辑
  • 结合两者生成直接查询 ODPS 原始表的新脚本，跳过中间表

快速开始

1

前往 配置 页面设置数据库连接和 API 密钥

2

在 可视化脚本 页面导入或创建脚本

3

使用 脚本转换 功能进行转换

4

通过 验证 功能确认转换结果
