数据库方言管理

数据库方言用于指定 SQL 脚本的语法规则。系统支持多种数据库方言，创建转换任务时需要为三个脚本分别指定方言。

当前支持的方言

  • MySQL - MySQL 数据库方言
  • 达梦 - 达梦数据库方言
  • PostgreSQL - PostgreSQL 数据库方言
  • ORACLE - Oracle 数据库方言
  • ODPS - 阿里云 ODPS (MaxCompute) 数据库方言

方言管理功能

  • 查看已启用的方言列表
  • 启用/禁用特定方言
  • 调整方言显示顺序
  • 添加新的方言（后续扩展）

LLM API 配置

LLM（大语言模型）用于 AI 智能转换功能。配置 API 密钥后，系统可以：

  • 智能分析 SQL 脚本结构和依赖关系
  • 理解中间表脚本的构建逻辑
  • 生成直接查询原始表的新脚本
  • 支持 SSE 流式输出转换过程

越低越确定，越高越随机

匿名化配置

匿名化功能用于对 SQL 脚本中的表名和字段名进行脱敏处理。配置排除列表可以指定不需要匿名化的字符串。

配置说明

  • 排除列表：逗号分隔的字符串列表，包含这些字符串的表名或字段名不会被匿名化
  • 大小写不敏感：判断时不区分大小写
  • 示例：如果排除列表包含 "date"，则 "create_date", "update_time" 等名称不会被匿名化
